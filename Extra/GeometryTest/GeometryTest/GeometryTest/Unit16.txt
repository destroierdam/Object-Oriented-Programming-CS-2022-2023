уравнение на множество спрямо афинна координатна система - Нека A е n-мерно афинно пространство, моделирано върху линейното пространство U, K = Oe1 ... en е афинна координатна система в A и B е подмножество на A. Нека S е някакво множество и f, g : Rn -> S. Ако P(x1, ..., xn) <- B <=> f(x1, ..., xn) = g(x1, ..., xn), то казваме, че B има спрямо K уравнение f(x1, ..., xn) = g(x1, ..., xn) и пишем B : f(x1, ..., xn) = g(x1, ..., xn). 
(скаларни) параметрични уравнения на множество спрямо афинна координатна система - Нека Л е някакво множество и h = (h1, ..., hn) : Л -> Rn . Ако P(x1, ..., xn) <- B <=> съществува л <- Л : xi = hi(л), i = 1, ..., n, то казваме, че xi = hi(л), i = 1, ..., n, са (скаларни) параметрични уравнения на B спрямо K и пишем B : { x1 = h1(л), ..., xn = hn(л)}, л <- Л, или накратко във векторна форма B : x = h(л), л <- Л
векторно параметрично уравнение на множество спрямо афинна координатна система - Нека Л е някакво множество и ~h : Л -> U. За P <- A означаваме r = OP (тоест r е радиус-векторът на P спрямо K). Ако P <- B <=> съществува л <- Л : r = ~h(л), то казваме, че r = ~h(л) е векторно параметрично уравнение на B спрямо K и пишем B : r = ~h(л), л <- Л.
